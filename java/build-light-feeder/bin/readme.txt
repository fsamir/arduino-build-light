Java COMM example
======================
List comm ports.

Java COMM API implementation uses rxtx library.


http://www.rxtx.org/
http://java.sun.com/products/javacomm/index.jsp


Compilation, jarring, running
=============================
* update setenv.bat to point your JDK folder.
* run compile.bat
* run jar.bat
* run go.bat
